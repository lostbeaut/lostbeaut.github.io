    line_number: true
    auto_detect: true

    主题_config.yml
    highlight_theme: normal

  4. 更换SITE-TITLE及MENU字体颜色
    找到博客目录下D:\Program Files\Blog\themes\next\source\css_common\outline\header\site-meta.styl文件，在.brand{}里添加以下代码：
    color: #ffb6c1 !important;

  5. 更改底部字体颜色
    找到博客目录下theme\next\source\css_custom\custom.styl，往里面添加以下代码：
    .footer-inner {color: #ffb6c1;}

  6. 修改左侧菜单字体颜色,比如 首页 字体颜色
    找到博客目录下D:\Program Files\Blog\themes\next\source\css_common\outline\header\menu.styl文件，在a{}里面添加以下代码：
    color: #8A2BE2 !important;
